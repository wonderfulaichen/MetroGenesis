# PolisCraft「改姓」计划 — 交接文档

## 核心策略

将 BlockUI / Structurize / Domum Ornamentum / MineColonies 开源代码**直接复制 + 包名重命名**嵌入 PolisCraft mod jar，确保 Forge / 网易双平台兼容。

## 包名映射

| 原包名 | 目标包名 |
|--------|----------|
| `com.ldtteam.blockui` | `com.poliscraft.blockui` |
| `com.ldtteam.structurize` | `com.poliscraft.structurize` |
| `com.ldtteam.domumornamentum` | `com.poliscraft.domumornamentum` |
| `com.minecolonies` | `com.poliscraft.minecolonies` |
| `com.ldtteam.common.*` | `com.poliscraft.blockui.*`（部分需独立路径） |

## 文件总数

- BlockUI 自研版（已改）：56 个
- BlockUI 缺失需补充：~30 个（hooks/、fakelevel/、language/、network/、support/ 等）
- Structurize：222 个
- Domum Ornamentum：136 个
- MineColonies：2064 个
- **合计：~2478 个 Java 文件**

## 执行步骤（建议顺序）

1. **补齐 BlockUI 缺失文件**：从 `参考/BlockUI-source/` 复制 hooks/、common/ 等模块
2. **批量包名替换**：用 sed 对 2478 个文件执行 `com.ldtteam.blockui` → `com.poliscraft.blockui` 等映射
3. **调整资源路径**：`assets/structurize` → `assets/poliscraft`、`assets/minecolonies` → `assets/poliscraft_mc` 等
4. **清理自研冗余**：删除旧 BOWindowBase.java（已被参考版 BOWindow 替代）、PolisLoader.java 等
5. **gradle build 验证**
