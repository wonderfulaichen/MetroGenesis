## 不可触碰约束（贯穿所有文档）

- `build.gradle` 的嵌入 4 包排除规则
- `Constants.MOD_ID = "minecolonies"`
- 嵌入第三方包名（minecolonies / structurize / domumornamentum / blockui）
- domumornamentum 为建筑方块内容，**用户明确不可删**
