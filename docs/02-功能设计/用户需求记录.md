## 核心愿景（北极星）

**"做自己的模组，把参考模组消化为 MetroGenesis 自有内容。"**

- MineColonies 是"素材库"而非"底座" → 提取模块 → 改姓嵌入 → 消化吸收为自有 RTS 机制。
- 用户目标 = MC 本地化的模拟城市模组，而非仅做道路特性。

## 技术决策记录

### UI 风格
- **配色方案A**（已定稿）：琥珀/橙色系（工业橙 `#D9772B` + 黄铜 `#C89B3C` + 金色粒子 `#E0BE5A`）。
- **布局**：Playlist 风格（左侧导航 + 右侧内容 + 顶部状态栏）。
- **设计规范**：`docs/01-架构设计/UI设计规范.md` v1.2 定稿。

### 架构约束
- `Constants.MOD_ID` 必须保持 `"minecolonies"`（否则 NoClassDefFoundError）。
- 嵌入包（minecolonies/structurize/domumornamentum/blockui）不可删除，但 minecolonies 为"死重"（2091 文件仅 1 处 import）。
- domumornamentum 是建筑方块（用户明确：不可删）。

### 执行铁律
R0 唤醒每日循环 → R1 市民生成 → R2 经济输入 → Phase 3 P0 区域生长（顺序不可乱）。
