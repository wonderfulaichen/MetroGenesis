# MetroGenesis UI 重建设计规范 — Create 像素工业风 v1

> 版本：v1.2 | 日期：2026-07-08 | 状态：✅ 定稿（概念图已确认）
> 决策来源：用户确认 — 整体风格走 **Create 机械动力风格 + 像素风**，参考 **Factorio / Create（工业机械）** 与 **Cities: Skylines / SimCity（现代城建信息架构）**；本次先出规范+概念图，实施范围稍后定。
> **新增参考**：Playlist 模组（https://modrinth.com/mod/playlist）的 UI 设计——半透明深色玻璃拟态、像素字体、强调色系统、极简控件，与目标风格高度契合。

---

## 一、设计目标与原则

1. **统一，而非并存**：当前存在三套互相矛盾的风格（文档定的牛皮纸工业风、早期科技蓝紫黑、代码实际跑的灰玻璃态），本次重做以本规范为唯一基线，消除分歧。
2. **暗色极简 + 像素 + 工业**：Mod 是"城市建造 RTS"，视觉基调采用 **Create 机械动力风格 + 像素风**——极深色背景、极细黄铜线条、像素化元素、金色粒子点缀，既工业又 Minecraft 原生。
3. **信息密度向 Cities: Skylines 看齐**：清晰的状态条、卡片化信息、数据可视化（图表/进度条），但以暗色极简呈现。
4. **可落地（纯 Java 绘制）**：Minecraft Forge 1.20.1，无外部 UI 库；质感用程序化绘制（线条、叠加噪点、渐变）模拟，不依赖外部贴图资源（贴图可后续补强）。

## 二、风格定位

| 维度 | 取法 | 说明 |
|------|------|------|
| 底色 | **半透明深色玻璃拟态** | 深色半透明背景（`#1a1a1a`~`#2a2a2a`，80%~90% 不透明度），与游戏场景融合（参考 Playlist 模组） |
| 框架/装饰 | **极细黄铜线 + 像素** | 1px 黄铜色线条边框（非粗框+铆钉），像素化图标与文字 |
| 质感 | **金色粒子 + 强调色** | 散落的金色像素光点；强调色 `#D9772B`（工业橙），用于高亮/选中/进度 |
| 信息架构 | Cities: Skylines + Playlist | 左侧导航栏 + 右侧内容区 + 顶栏状态条 + 底栏工具栏 |
| 数据呈现 | Cities: Skylines | 进度条、微型图表、明确数字与单位（C-Value） |
| 控件风格 | Playlist 模组 | 像素字体、圆角矩形按钮、深色底+浅色文字、右键上下文菜单 |

**一句话基调**："Playlist 风格的城市政控台"——半透明深色玻璃底，黄铜细线为骨，像素为形，金色粒子点缀。

## 三、配色系统

### 3.1 基础底色（半透明深色玻璃拟态）

```java
// 主背景（半透明深色，与游戏场景融合）
C_BG            = 0xCC1A1A1A;   // #1A1A1A 80% 不透明度，主背景
C_BG_PANEL      = 0xDD1E1E1E;   // #1E1E1E 87% 不透明度，面板背景
C_BG_CARD       = 0xE62A2A2A;   // #2A2A2A 90% 不透明度，卡片背景
C_BG_HOVER      = 0xF0333333;   // #333333 94% 不透明度，悬停/选中
C_BG_MODAL      = 0xF01A1A1A;   // #1A1A1A 94% 不透明度，模态窗口背景（更深）
```

### 3.2 黄铜（Brass，细线条+强调主色）

```java
C_BRASS         = 0xC89B3C;   // #C89B3C 主黄铜（边框、标题）
C_BRASS_LT      = 0xE0BE5A;   // #E0BE5A 黄铜高光（悬停）
C_BRASS_DK      = 0x9C7728;   // #9C7728 黄铜阴影
```

### 3.2.1 强调色系统（琥珀/橙色系）

> 与黄铜同属暖色系，工业感强，像熔炉/铜管的温暖光泽。

```java
C_ACCENT        = 0xD9772B;   // #D9772B 工业橙（主强调色）
C_ACCENT_DK     = 0xB35E1F;   // #B35E1F 橙色阴影
C_ACCENT_LT     = 0xE8944A;   // #E8944A 橙色高光
```

### 3.3 绿色（仅用于成功状态）

> 绿色降级为功能色，不再用于粒子效果。

```java
C_GREEN_SUCCESS = 0x5A8A4A;   // #5A8A4A 苔藓绿（成功/完成状态）
C_GREEN_DK      = 0x3A6A2A;   // #3A6A2A 深苔藓绿（次要）
```

### 3.4 功能/行动色

```java
C_ACTION        = 0xD9772B;   // #D9772B 工业橙（主行动/确认/进行中）
C_WARNING       = 0xE8B53A;   // #E8B53A 警示黄
C_DANGER        = 0xA64B32;   // #A64B32 砖红（危险/删除/赤字）
```

### 3.4.1 金色/琥珀粒子系统

> 替代绿色粒子，与黄铜边框协调，像熔炉火花/铜管光泽。

```java
C_PARTICLE      = 0xE0BE5A;   // #E0BE5A 金色粒子（主粒子颜色）
C_PARTICLE_DK   = 0xC89B3C;   // #C89B3C 黄铜色粒子（次要）
C_PARTICLE_LT   = 0xF0D878;   // #F0D878 金色高光粒子
```

### 3.5 文字色（浅色文字，配极深背景）

```java
C_TEXT          = 0xE0D8CC;   // #E0D8CC 主文字（暖白）
C_TEXT_2ND      = 0x9A9088;   // #9A9088 次要文字（暖灰）
C_TEXT_3RD      = 0x6A6058;   // #6A6058 三级/禁用
C_TEXT_BRASS    = 0xC89B3C;   // #C89B3C 黄铜色文字（标题/数字强调）
```

### 3.6 功能区色（住宅/工业/商业/农业/公共）

> 压低饱和，适配极深色背景。黄铜为主调，其余色偏暗。

```java
ZONE_RESIDENTIAL = 0x4A8A4A;   // 住宅 暗绿
ZONE_INDUSTRIAL  = 0xB5703A;   // 工业 锈橙
ZONE_COMMERCIAL  = 0x5A7A8A;   // 商业 雾蓝
ZONE_AGRICULTURE = 0x7A8A3A;   // 农业 黄绿
ZONE_PUBLIC      = 0x7A6A8A;   // 公共 淡紫
```

| **细分割线** | 1px `#333333` 水平线，两端留 8px 边距 | 区域分隔 |

## 五、字体与文字层级

- **字体**：使用 Minecraft 默认字体（Forge 不支持换字体文件）；中文用系统回退。不可改字型，但可模拟像素风——用**颜色+粗细+像素化图标**表达层级。
- 像素风增强：关键数字（资金/人口）用黄铜色 `C_TEXT_BRASS`；标签用橙色 `C_ACCENT`；图标用像素化方块拼（1px~2px 单位）。
- 层级：
  - 标题：16px，`C_TEXT_BRASS`，粗体（§l）
  - 主数据：14px，`C_TEXT`
  - 次要：12px，`C_TEXT_2ND`
  - 提示/禁用：11px，`C_TEXT_3RD`
  - 数字强调（资金/人口）：14px，`C_TEXT_BRASS`
  - 标签：11px，`C_ACCENT`（橙色像素标签）

## 六、布局原则（HUD 信息架构）

参考 Cities: Skylines + Playlist 模组的布局，统一所有界面：

```
┌──────────────────────────────────────────────────────────┐
│ [顶栏状态条] 城邦名 | 资金💰 | 人口👥 | 满意度😊 | ×   │  ← 半透明深色+黄铜细线
├──────────────┬───────────────────────────────┬───────────┤
│ [左侧导航栏] │   [中央 3D 城市俯视/工作区]    │ [右侧详情]│  ← Playlist 风格
│ 主界面       │   （游戏世界可见，黄铜窗框）   │ 建筑/市民 │
│ 建筑图鉴     │                               │           │
│ 市政厅       │                               │           │
│ 建造工具     │                               │           │
│ 市民管理     │                               │           │
├──────────────┴───────────────────────────────┴───────────┤
│ [底栏工具栏] 住宅 | 工业 | 商业 | 农业 | 公共 | 菜单     │  ← 半透明深色+强调色高亮
└──────────────────────────────────────────────────────────┘
```

- **Playlist 风格左侧导航栏**：固定宽度 120px，半透明深色背景，白色像素文字，选中项用强调色 `C_ACCENT` 高亮。图标使用黄铜色像素图标。
- **半透明玻璃面板**：所有面板使用 §3.1 半透明深色底，与游戏场景融合。
- **统一边距**：面板内边距 8px，元素间距 4px。
- **统一圆角**：所有面板/按钮 4px 圆角（用 `fillRoundRect` 思路或四角削角）。
- **统一栅格**：工具栏/卡片按固定栅格对齐，避免自由散落。
- **像素化边角**：四角用 2x2 像素方块（黄铜色）代替圆角/铆钉，增强像素感。

### 6.1 面板布局原则（基于第一张概念图确认）

> 保持与第一张概念图一致的布局风格。

1. **主内容区结构**：
   - **顶部**：大标题（白色像素字体）+ 橙色下划线/进度条
   - **中部**：主要内容区（卡片网格/信息展示）
   - **底部**：状态信息栏（关键数据卡片）

2. **信息卡片设计**：
   - 深色半透明底（`C_BG_CARD`）
   - 黄铜细边框（1px `#C89B3C`）
   - 像素字体文字（白色为主，橙色强调数字）
   - 卡片内边距 8px，卡片间距 4px

3. **进度条/状态指示**：
   - 顶部状态栏：深灰底 + 橙色填充 + 白色文字（如"LOADING"）
   - 进度条：深灰凹槽 + 橙色填充（`C_ACCENT`）
   - 百分比数字：橙色强调

4. **粒子效果**：
   - 金色/橙色粒子（`C_PARTICLE`）漂浮在主内容区
   - 1~2px 方块，20%~50% 透明度，缓慢移动
   - 集中在内容区中部，不遮挡文字

## 七、组件规范

### 7.1 按钮（Playlist 风格）
- **普通按钮**：深灰半透明底 `#2A2A2A`（90% 不透明度）+ 黄铜细边框 `#C89B3C`（1px）+ 圆角 2px + 白色像素文字。
- **强调按钮**：强调色底 `C_ACCENT`（`#D9772B` 工业橙）+ 白色文字，用于主要操作（确认/建造）。
- **悬停**：底变 `#333333`，黄铜边变 `#E0BE5A`（发光感）。
- **按下**：整体下移 1px + 底变 `#1A1A1A`（凹陷感）。
- **禁用**：`#1A1A1A` 底 + `#6A6058` 文字，无像素点缀。
- **图标按钮**：仅图标无文字，用于工具栏/播放控制（参考 Playlist 播放器）。

### 7.2 面板/卡片（半透明玻璃拟态）
- **半透明玻璃面板**：底 `C_BG_PANEL`（半透明深色）+ 黄铜细边框（1px）+ 圆角 2px；用于所有主面板。
- **深灰卡片**：底 `C_BG_CARD`（半透明深色）+ 黄铜细边框（1px）+ 四角 2x2 像素方块；用于建筑图鉴条目、信息卡。
- **模态窗口**：底 `C_BG_MODAL`（更深半透明）+ 黄铜边框 + 标题栏 `< 标题`；用于设置、确认对话框（Playlist 风格）。
- 所有卡片/面板背景可叠加微弱金色粒子效果（`#E0BE5A`，10% 透明度）增加层次感。

### 7.3 进度条/滚动条（Playlist 风格）
- **进度条**：深灰凹槽 `#1A1A1A` + 强调色填充 `C_ACCENT`（`#D9772B` 工业橙）；用于扫描进度、建设进度。
- **音量条/垂直滑块**：参考 Playlist 像素风格，橙色+深蓝色渐变，顶部显示百分比。
- **滚动条**：极细黄铜色滑块 `#C89B3C`，宽度 4px。
- 顶部可叠加黄黑警示条纹（Factorio 风），增强"工程进度"语义。

### 7.4 上下文菜单（Playlist 风格）
- **右键菜单**：深色半透明底 `#1A1A1A`（94% 不透明度）+ 白色像素文字 + 1px 黄铜边框，圆角 2px。
- **菜单项**：悬停高亮 `#333333`，选中项用强调色 `C_ACCENT`（`#D9772B` 工业橙）。
- **分隔线**：1px `#333333` 水平线。
- **图标**：菜单项左侧可带像素图标（播放、添加、移除等）。

### 7.5 开关控件（Playlist 风格）
- **开关**：深色底 `#1A1A1A` + 白色/橙色滑块，圆角矩形。
- **状态**：关闭=灰色滑块，开启=强调色滑块 `C_ACCENT`（`#D9772B` 工业橙）。
- **标签**：左侧白色像素文字标签。

### 7.6 下拉菜单（Playlist 风格）
- **触发器**：深灰半透明底 + 白色文字 + 黄铜边框，圆角 2px。
- **下拉列表**：半透明深色底 + 白色文字，选中项用强调色 `C_ACCENT`（`#D9772B` 工业橙）高亮。
- **图标**：右侧小箭头指示。

### 7.7 图标（黄铜线条像素图标）
统一用黄铜色单色线性图标（避免彩色 emoji 杂乱），像素化风格：
- 资金：硬币/钱袋 ｜ 人口：小人 ｜ 满意度：笑脸/表盘
- 住宅：房屋 ｜ 工业：齿轮/工厂 ｜ 商业：商店 ｜ 农业：麦穗 ｜ 公共：市政厅/柱
- 旋转：环形箭头 ｜ 镜像：对称箭头 ｜ 确认：对勾 ｜ 取消：叉
- 播放：三角形 ｜ 暂停：双竖线 ｜ 上一曲/下一曲：双箭头（Playlist 风格）

### 7.8 数据可视化
- 满意度/需求：用黄铜环形或水平分段条。
- 财政：收入用金色小箭头 + 数字，支出用砖红小箭头 + 数字。
- 区划：地图色块用 §3.6 功能区色，叠加深灰半透明蒙版统一调性。

### 7.9 信息卡片（基于第一张概念图确认）

> 用于展示关键数据（人口、满意度、资源等）。

- **布局**：底部状态栏，2~3 个卡片水平排列
- **尺寸**：宽度 150~200px，高度 60~80px
- **样式**：
  - 背景：`C_BG_CARD`（半透明深色）
  - 边框：1px `#C89B3C` 黄铜细线
  - 圆角：2px
  - 内边距：8px
- **内容**：
  - 标题：白色像素字体，12px
  - 数值：橙色强调 `C_ACCENT`，14px
  - 单位/标签：白色像素字体，10px
- **示例**：
  ```
  ┌─────────────┐
  │ CITY        │
  │ CITIZENS 42 │
  │             │
  └─────────────┘
  ```

## 九、代码落地指引（实施阶段用，本次仅规范）

1. **抽取样式常量类** `com.metrogenesis.gui.MGStyles`：集中 §3 全部色值 + `drawBrassFrame()` / `drawPixelCorner()` / `drawDarkBg()` / `drawGreenParticles()` / `drawHazardStripe()` 等绘制工具方法。
2. **半透明玻璃拟态实现**：使用 `fillGradient` 或 `fill` 绘制半透明矩形，叠加模糊效果（可选）。参考 Playlist 模组的 `Darken Background` 选项。
